## Why

Golang 系列（一至十二）完成了基礎到實戰再到面試/陷阱的學習路徑，但讀者在實際工作中還需要掌握進階工具和架構模式。Web 框架、ORM、PostgreSQL、gRPC、微服務和 Kubernetes 是 Go 生態中最重要的進階主題，補齊這六篇能讓系列從「入門」延伸到「職場即戰力」。

## What Changes

- 新增六篇進階 Go 文章，延續系列編號（十三至十八）
- 第十三篇：Web 框架（Gin vs Echo）— 對比標準庫，了解框架提供的價值
- 第十四篇：ORM 與資料庫工具（GORM + sqlx）— 簡化資料庫操作
- 第十五篇：PostgreSQL 進階 — 從 SQLite 升級，學習 migration、index、transaction
- 第十六篇：gRPC — Protocol Buffers、服務定義、streaming
- 第十七篇：微服務架構 — 服務拆分、通訊模式、服務發現
- 第十八篇：Kubernetes 部署 — 容器編排、Deployment、Service、ConfigMap
- 更新第十二篇末尾，加入進階系列導覽

## Capabilities

### New Capabilities
- `golang-gin-echo-post`: Web 框架介紹，Gin 與 Echo 的對比、路由、中間件、錯誤處理
- `golang-orm-post`: GORM 與 sqlx 的使用，Model 定義、CRUD、Migration、關聯查詢
- `golang-postgresql-post`: PostgreSQL 進階操作，連線池、Migration 工具、Index、Transaction、JSON 欄位
- `golang-grpc-post`: gRPC 開發，Protocol Buffers、Unary/Streaming RPC、攔截器
- `golang-microservices-post`: 微服務架構，服務拆分策略、HTTP/gRPC 通訊、服務發現、分散式追蹤
- `golang-kubernetes-post`: Kubernetes 部署，Dockerfile 最佳化、K8s 資源定義、Helm、健康檢查
