## Why

部落格目前缺乏 Go 語言的系列教學內容。Go 是現今後端開發最受歡迎的語言之一，從基礎語法到實戰建構 Web 應用程式的完整系列文章，能為讀者提供循序漸進的學習路徑，同時提升部落格在技術內容的深度與廣度。

## What Changes

- 新增一系列 Go 語言教學文章（預計 8-10 篇），涵蓋從零開始到建構完整 Web 應用程式
- 系列文章採用循序漸進的結構，每篇文章都建立在前一篇的基礎上
- 所有文章歸類於「技術筆記」分類，使用 `golang` 和 `webapp` 等相關標籤
- 最終目標：讀者能從零開始，跟著系列文章建構出一個具備 CRUD、認證、資料庫的完整 Web 應用程式

## Capabilities

### New Capabilities
- `golang-fundamentals-posts`: Go 語言基礎篇文章（語言介紹、環境設定、基礎語法、資料結構、併發）
- `golang-webapp-posts`: Go Web 應用實戰篇文章（HTTP 基礎、路由、資料庫、認證、部署）

### Modified Capabilities

## Impact

- 新增 8-10 個 markdown 文章檔案於 `content/posts/` 目錄
- 可能需要新增範例程式碼的圖片或輔助資源於 `static/images/`
- 不影響現有文章或網站配置
